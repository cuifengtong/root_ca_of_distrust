# 说明
本项目旨在让大家认识到CNNIC以及其他流氓根证书的危害并让大家有办法禁用它们。目前本项目中收入了3个根证书。它们是：
+ CNNIC Root CA
+ CNNIC DQ CA
+ Alipay Root CA  
如果大家发现了其他的流氓CA，欢迎发布PR/ISSUE来反馈！

# 使用
这里不介绍了，自己来[我的博客](https://aluoblog.wordpress.com/2018/01/20/how-to-distrust-cnnic-ca/)里看吧。
